Quest Actions Test

This page tests the quest start and delete functionality.

Test Instructions

  1. Open the browser developer tools (F12)
  2. Go to the Network tab
  3. Navigate to the quest dashboard in the app
  4. Try to start a quest (click the Start button on a draft quest)
  5. Try to delete a quest (click the Delete button on a draft quest)
  6. Check the Network tab for API calls to verify they're being made

Expected API Calls

When you start a quest, you should see:

POST /quests/quests/{quest_id}/start

When you delete a quest, you should see:

DELETE /quests/quests/{quest_id}

When you cancel a quest, you should see:

POST /quests/quests/{quest_id}/cancel

When you fail a quest, you should see:

POST /quests/quests/{quest_id}/fail
